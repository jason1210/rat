
# rat
十二生肖-鼠（Rat）
#十二生肖（Zodiac）：
鼠、牛、虎、兔、龙、蛇、马、羊、猴、鸡、狗、猪
Rat, cow, tiger, rabbit, dragon, snake, horse, sheep, monkey, chicken, dog, pig

### 组织结构

SpingBoot+Mybatis+Mysql+Dubbo+RabbitMQ

``` 
rat
├── rat-api -- 服务api
├── rat-server -- 服务
├── rat-admin -- 后台管理系统
├── rat-portal -- 门户系统
├── rat-web -- 门户页面 ，基于vue
└── rat-demo -- 框架搭建时的测试代码
```
